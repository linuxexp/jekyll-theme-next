If you want to test something on internet explorer in linux, then you may think of wine, but here's why you shouldn't do that... The inbuild wine's windows explorer uses the gecko engine, the same engine that firefox uses, so in a sense if you are thinking of testing some web developement stuff in wine's internet explorer it won't worth the time, the rendering will be same as firefox.. So what, does that mean you are left hopeless.. Below are some solution.

-> Try ies4linux, http://www.tatanka.com.br/ies4linux/page/Main_Page, in my case it didn't worked out, the links hardcoded inside the installer seems to be broken along with other critical bugs..

-> Install actual internet explorer inside wine with winetricks...
Following instructions are given to install internet explorer 6 inside wine, here's how to do that..
Make sure you have wine installed, for this post I installed wine 1.3

sudo apt-get install wine
cd Desktop
Get Winetricks
wget http://kegel.com/wine/winetricks
Download and Install ie6 Size ~~ 100 MBs
WINEARCH=win32 WINEPREFIX=~/.wine_ie6 sh winetricks -q ie6
Open IE6
WINEARCH=win32 WINEPREFIX=~/.wine_ie6 wine ~/.wine_ie6/drive_c/Program Files/Internet Explorer/IEXPLORE.EXE
