I was doing some developing work in windows which was running inside virtualbox, It was about manipulating critical system registry. As a result of incorrect settings, I wasn't able to login. So I needed to revert the settings, I thought to share the Idea here.

Mount VirtualBox Harddisk

First we need the qemu, you may know it for its processor emulation and stuff.

  • sudo apt-get install qemu-nbd
  • modprobe nbd
  • qemu-nbd -c /dev/nbd0 <harddisk-image-vdi-file>
  • mount /dev/nbd0p1 /mnt

Now the harddisk is available as normal storage mount

Editing registry from linux with chntpw

for example to edit HKLMSoftwareMicrosoftWindows NTCurrentVersionWinlogon, key "Userinit" value

  • chntpw -e /mnt/WINDOWS/system32/config/<System, Software...>
  • cd MicrosoftWindows NTCurrentVersionWinlogon
  • ed Userinit

now change the value to whatever you want then

  • q
  • yes
