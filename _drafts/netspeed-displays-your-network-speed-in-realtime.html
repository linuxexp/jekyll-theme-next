The content of this post is actually a re-post, since I deleted my old blog. So here it is a simple, tiny gtk based utility for displaying realtime network speed of all network interfaces in realtime.. The application counts the received bytes on each network interface.. The application puts no load over bandwidth, you could have it running (tip: right click and keep it on top), it'll display the network speed at which you are receiving data from network.
